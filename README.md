# Livro Deitel Java Como Programar - 10ª Edição

Bem-vindo ao meu repositório de estudos e práticas do livro "Java Como Programar" de Deitel, na 10ª edição.

### Descrição

O livro abrange os princípios fundamentais da programação em Java, fornecendo uma base sólida para o desenvolvimento de software. 
Este repositório é uma compilação de meus esforços para compreender e aplicar os conceitos apresentados no livro.


## Objetivo

Meu objetivo ao ao aplicar as praticas contidas no livro é aperfeiçoar conceitos já conhecidos e absorver novas práticas e conteudos, para assim melhor avançar na carreira.

### Estrutura do Repositório
